# stateCompletionRate

Source code application for Topic 3.

## Installation

Open a terminal in the project root directory and install the required dependencies: 

```bash
pip install streamlit scikit-learn pandas numpy plotly scipy statsmodels kneed tslearn matplotlib
```
If you encounter permission issues, run the terminal as Administrator or use:
```bash
pip install --user streamlit scikit-learn pandas numpy plotly scipy statsmodels kneed tslearn matplotlib
```

## Running the Application

Navigate to the project's root directory and run:

```bash
streamlit run education_analysis_app.py
```

The application will start and open in your default web browser.
